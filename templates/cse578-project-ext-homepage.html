Summary:


This homepage is for the extended visualization for rendering tSNE of the financial data. The flask application processes the financial data to create a plot of similar data forming in clusters based on the selected columns or features of the initial data set. The data is imported into a dataframe, scaled, and then fitted with a tSNE model. The scaled data is then used to generate output with x,y format. The latter is retrieved and rendered into d3 using an HTTP POST request which is handled for updates.

Precondition: the flask instance of application.py must be running prior to analysis with web app navigation.
